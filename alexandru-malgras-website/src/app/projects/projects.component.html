Model Validation 3D Concrete Printing

Type: Internship at Cybe Construction

Location: Oss, North Brabant, Netherlands | Remote

Date: Aug 2022 - Sep 2023


I developed a model validation service to guarantee the successful printing of 3D models. The validation involves a set of algorithms that enforce rules specific to 3D concrete printing.


The algorithms are automatically analysing a model when it is added to the software that is converting the model to printable code. The user can opt in or out to run the validation by interacting with the validation checkbox in the user interface. The algorithms are customizable based on the user needs. The user can customize the algorithms by interacting with the settings user interface. In the video, the algorithms are applied on different 3D models. The results can be seen in the console, and when applicable on the model directly. In examples 2 and 3, wall 3D models of 3m height and 3m width are validated by the model validator.


Skills: C#, Problem Solving, .Net WPF, Unit Testing, Git

Electric Powerboat Dashboard

Type: Internship at Edorado Marine

Location: Dordrecht, South Holand, Netherlands | Remote

Date: Feb 2021 - Jul 2021


At Edorado Marine, I utilized the Elastic Stack to create a dashboard on Amazon Web Services that enables engineers to analyze data more efficiently. The dashboard displays boat-related data collected through a custom logging program that I developed. This program directly retrieves the data from Docker for easy and streamlined access.


In the initial phase of the project, the task was to explore various data storage services available on AWS and identify the most appropriate one. Afterward, the focus was on configuring the selected service, developing a logging program, and presenting essential data and graphs on the frontend.


Skills: Java, AWS Cloud, ELK Stack, S3, SQL, Git, Docker
